# Mops Agent

## Get Started

The agent requires a `GOOGLE_API_KEY` environment variable to be set.

### 1. Run Dev UI

```shell
bazel run //projects/mops/mops_agent/src/main/java/lab/mops/agent/multitool
```

### 2. Talk to agent

- Open http://localhost:8888 and interact with the agent.
