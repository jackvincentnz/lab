# Glossary

## Category

A user defined category that contains values that can categorize entities in the system.

### Example

```text
Name: Country
Values: [
  New Zealand,
  Canada
]
```

### Categorization

A categorization of an entity by attributing a category value from a category to it.

### Example

```text
Entity: LineItem
Categorization:
  Country => New Zealand
```
